飞雪连天射白鹿 笑书神侠倚碧鸳

发表于 2018-10-30 更新于 2019-10-17

笑傲江湖，侠影萍踪，几许英豪？

算八部天龙，逐鹿问鼎。

神雕侠侣，领袖群雄。

屠龙宝刀，倚天长剑，

且赠英雄射大雕。肝胆照，纵连城异宝，也愿全抛。

唯欲长天啸，问苍穹，此生今朝？

叹鸳鸯一梦，碧血脉脉。

书剑恩仇，飞雪飘飘。

曲终人散，侠客越女，尽化长江滚滚滔。

猛回头，看西风漫漫，白马萧萧。

没有金庸先生的大才，只能借来一篇以表我心

“今番良晤，豪兴不浅，他日江湖相逢，再当杯酒言欢。咱们就此别过。”

# 金庸 # 武侠
